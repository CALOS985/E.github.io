背景元素高级参数

这些参数能用于任何背景元素. (即 静态型背景元素 , 动画型背景元素 , 视差型背景元素 , 其他背景元素类型 .)

trans = trans ; (可选) (字符串型)
指定图形运用的透明混合效果.透明模式为default, none, add, sub.
这些分别表示 默认特征,没有透明效果,颜色叠加,颜色相减.
default和none都会引起非动画型元素没有透明混合效果.
对于动画型背景元素,指定default将使用动画指定的透明混合效果,而其他的透明值将覆盖动画中的透明标示.
如果指定了add,可能需要适用alpha参数来控制混合效果.默认值是default.

注意:如果mugen版本是1.0以下,对动画型元素来说default和none特征相同.

alpha = src_alpha,dst_alpha ; (可选) (整型,整型)
参见trans控制器相关参数.

id = id_number ; (整型)
给背景元素指定一个ID号.不同的元素可以共享相同ID.
ID号作用是允许背景控制器指定要操作哪个元素.缺省默认是0.

positionlink = link_flag ; (布尔型)
设置positionlink为1可以固定此元素的位置到DEF文件中紧挨着它的另一个元素的位置.
如果设置为1,当前元素start参数值被视为是与上一个元素位置的偏移量.
delta参数在此情况下降无效.用于使一大堆元素整齐一致的运动.
如果你编辑positionlink链中的第一个元素的start和delta值,效果将在整个链中的所有元素中都体现出来.
缺省默认为0.

maskwindow = x1,y1,x2,y2 ; (整型,整型,整型,整型) (1.1或以后版本)
给背景元素指定一个遮罩窗口.一个遮罩窗口是屏幕上一个包含此图像的概念矩形.
任何超出这个矩形的图像部分都不会显示出来.
x1,y1指定了窗口的左上角坐标,x2,y2 指定了窗口右下角坐标.此坐标相对于舞台原点中心(屏幕的顶部中心点).
缺省将不执行.

window = x1,y1,x2,y2 ; (整型,整型,整型,整型) (废弃)
此废弃参数仍可用,但不保证在缩放的时候准确性.
指定了一个相对于屏幕右上角坐标的遮罩窗口,端点也包含在内.

windowdelta = window_dx,window_dy ; (浮点型,浮点型)
指定遮罩窗口的delta运动.运行方式和背景元素自身的delta参数差不多.默认为0,0.

scalestart = scale_x, scale_y ; (浮点型,浮点型)
初始缩放值.默认为1,1.

scaledelta = scale_dx, scale_dy ; (浮点型,浮点型)
和delta参数相似,这是摄像机每移动1个单位元素缩放量.默认为0,0.

velocity = vel_x, vel_y ; (浮点型,浮点型)
指定背景元素初始x,y方向速度.(默认为0)
此功能在背景控制器VelSet中也包含.

zoomdelta = dzoom ; (浮点型)
指定摄像机缩放影响元素缩放的量.0表示元素在缩放时不改变尺寸,1表示元素在缩放时和摄像机缩放系数相同.默认为1.

sin.x = amplitude, period, phase ; (浮点型,浮点型,浮点型)
指定元素在x方向的正弦运动.
第1个参数是振幅,第2个参数是运动周期(帧数),第3个参数指定初始正弦运动的相位偏移.(默认是0,例如,元素将正弦范围正中时开始运动)
这参数基本上被SinX背景控制器取代了.

sin.y = amplitude, period, phase ; (浮点型,浮点型,浮点型)
运行方式和sin.x相同,只是y方向.
