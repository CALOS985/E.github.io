Const (*)

返回玩家的一些常数.

格式:
Const(参数名)

参数:
参数名
需要检测的常数名.有效参数为:

返回值类型:
取决于指定的打击参数.查看详细信息.

错误条件:
无

详细信息:
下面参数名的值将返回玩家的cns中[Data]组中指定的值.

  • data.life:返回"life(生命值)"参数值. (整型)
  • data.power:返回"power(能量值)"参数值. (整型)
  • data.attack:返回"attack(攻击力)"参数值. (整型)
  • data.defence:返回"defence(防御力)"参数值. (整型)
  • data.fall.defence_mul:返回defence multiplier(防御系数值),公式为 100/(f+100),这里的f指的是"fall.defence_up"参数.(浮点型)
  • data.liedown.time:返回"liedown.time(倒地起身时间)"参数值. (整型)
  • data.airjuggle:返回"airjuggle(控制连击的参数)"参数值. (整型)
  • data.sparkno:返回"sparkno(火花号)"参数值. (整型)
  • data.guard.sparkno:返回"guard.sparkno(防御火花号)"参数值. (整型)
  • data.KO.echo:返回"KO.echo(被KO时有没有回声)"参数值. (整型)
  • data.IntPersistIndex:返回"IntPersistIndex(控制var参数初始化)"参数值. (整型)
  • data.FloatPersistIndex:返回"FloatPersistIndex(控制fvar参数初始化)"参数值. (整型)

下面参数名的值将返回玩家的cns中[Size]组中指定的值.

  • size.xscale:返回"xscale(x方向缩放)"参数值. (浮点型)
  • size.yscale:返回"yscale(y方向缩放)"参数值. (浮点型)
  • size.ground.back:返回"ground.back(后方宽度)"参数值. (整型)
  • size.ground.front:返回"ground.front(前方宽度)"参数值. (整型)
  • size.air.back:返回"aie.back(空中后方宽度)"参数值. (整型)
  • size.air.front:返回"aie.front(空中前方宽度)"参数值. (整型)
  • size.height:返回"height(高度)"参数值. (整型)
  • size.attack.dist:返回"attack.dist(攻击距离,对手超出这个距离无法做出防御动作)"参数值. (整型)
  • size.proj.attack.dist:返回"proj.attack.dist(飞行道具攻击距离)"参数值. (整型)
  • size.proj.doscale:返回"proj.doscale(飞行道具是否受size.xscale和size.yscale影响)"参数值. (整型)
  • size.head.pos.x:返回"head.pos(头部位置)"x方向参数值. (整型)
  • size.head.pos.y:返回"head.pos(头部位置)"y方向参数值. (整型)
  • size.mid.pos.x:返回"mid.pos(身体中部位置)"x方向参数值. (整型)
  • size.mid.pos.y:返回"mid.pos(身体中部位置)"y方向参数值. (整型)
  • size.shadowoffset:返回"shadowoffset(影子偏移)"参数值. (整型)
  • size.draw.offset.x:返回"draw.offset(人物偏移)"x方向参数值. (整型)
  • size.draw.offset.y:返回"draw.offset(人物偏移)"y方向参数值. (整型)

下面参数名的值将返回玩家的cns中[Velocity]组中指定的值.

  • velocity.walk.fwd.x:返回"walk.fwd(向前走速度)"参数值. (浮点型)
  • velocity.walk.back.x:返回"walk.back(向后走速度)"参数值. (浮点型)
  • velocity.run.fwd.x:返回"run.fwd(向前跑)"x方向参数值. (浮点型)
  • velocity.run.fwd.y:返回"run.fwd(向前跑)"y方向参数值. (浮点型)
  • velocity.run.back.x:返回"run.back(向后跑)"x方向参数值. (浮点型)
  • velocity.run.back.y:返回"run.back(向后跑)"y方向参数值. (浮点型)
  • velocity.jump.y:返回"jump.neu(正向上跳)"y方向参数值. 注意: 这不是"velocity.jump.neu.y". 仅仅是"neu"参数取y方向的值. (浮点型)
  • velocity.jump.neu.x:返回"jump.neu(正向上跳)"x方向参数值. (浮点型)
  • velocity.jump.back.x:返回"jump.back(向后跳)"参数值. (浮点型)
  • velocity.jump.fwd.x:返回"jump.fwd(向前跳)"参数值. (浮点型)
  • velocity.runjump.back.x:返回"runjump.back(向后跑跳)"参数值. (浮点型)
  • velocity.runjump.fwd.x:返回"runjump.fwd(向前跑跳)"参数值. (浮点型)
  • velocity.airjump.y:返回"airjump.neu(正向上连跳)"y方向参数值. 注意: 这不是"velocity.airjump.neu.y". (浮点型)
  • velocity.airjump.neu.x:返回"airjump.neu(正向上连跳)"x方向参数值. (浮点型)
  • velocity.airjump.back.x:返回"airjump.back(正向后连跳)"参数值. (浮点型)
  • velocity.airjump.fwd.x:返回"airjump.fwd(正向前连跳)"参数值. (浮点型)
  • velocity.air.gethit.groundrecover.x:返回"air.gethit.groundrecover"x方向参数值. (浮点型)
  • velocity.air.gethit.groundrecover.y:返回"air.gethit.groundrecover"y方向参数值. (浮点型)
  • velocity.air.gethit.airrecover.mul.x:返回"air.gethit.airrecover.mul"x方向参数值. (浮点型)
  • velocity.air.gethit.airrecover.mul.y:返回"air.gethit.airrecover.mul"y方向参数值. (浮点型)
  • velocity.air.gethit.airrecover.add.x:返回"air.gethit.airrecover.add"x方向参数值. (浮点型)
  • velocity.air.gethit.airrecover.add.y:返回"air.gethit.airrecover.add"y方向参数值. (浮点型)
  • velocity.air.gethit.airrecover.back:返回"air.gethit.airrecover.back"参数值. (浮点型)
  • velocity.air.gethit.airrecover.fwd:返回"air.gethit.airrecover.fwd"参数值. (浮点型)
  • velocity.air.gethit.airrecover.up:返回"air.gethit.airrecover.up"参数值. (浮点型)
  • velocity.air.gethit.airrecover.down:返回"air.gethit.airrecover.down"参数值. (浮点型)

下面参数名的值将返回玩家的cns中[Movement]组中指定的值.

  • movement.airjump.num:返回"airjump.num(空中连跳数)"参数值. (整型)
  • movement.airjump.height:返回"airjump.height(空中连跳需要的高度)"参数值. (整型)
  • movement.yaccel:返回"yaccel(下落加速度)"参数值. (浮点型)
  • movement.stand.friction:返回"stand.friction(站立受到的摩擦力)"参数值. (浮点型)
  • movement.crouch.friction:返回"crouch.friction(蹲下受到的摩擦力)"参数值. (浮点型)
  • movement.stand.friction.threshold:返回"stand.friction.threshold(站立摩擦力阈值)"参数值. (浮点型)
  • movement.crouch.friction.threshold:返回"crouch.friction.threshold(蹲下摩擦力阈值)"参数值. (浮点型)
  • movement.jump.changeanim.threshold:返回"jump.changeanim.threshold"参数值. (浮点型)
  • movement.air.gethit.groundlevel:返回"air.gethit.groundlevel"参数值. (浮点型)
  • movement.air.gethit.groundrecover.ground.threshold:返回"air.gethit.groundrecover.ground.threshold"参数值. (浮点型)
  • movement.air.gethit.groundrecover.groundlevel:返回"air.gethit.groundrecover.groundlevel"参数值. (浮点型)
  • movement.air.gethit.airrecover.threshold:返回"air.gethit.airrecover.threshold"参数值. (浮点型)
  • movement.air.gethit.airrecover.yaccel:返回"air.gethit.airrecover.yaccel"参数值. (浮点型)
  • movement.air.gethit.trip.groundlevel:返回"air.gethit.trip.groundlevel"参数值. (浮点型)
  • movement.down.bounce.offset.x:返回"down.bounce.offset"x方向参数值. (浮点型)
  • movement.down.bounce.offset.y:返回"down.bounce.offset"y方向参数值. (浮点型)
  • movement.down.bounce.yaccel:返回"down.bounce.yaccel"参数值. (浮点型)
  • movement.down.bounce.groundlevel:返回"down.bounce.groundlevel"参数值. (浮点型)
  • movement.down.friction.threshold:返回"down.friction.threshold"参数值. (浮点型)

示例:
trigger1 = Const(velocity.walk.fwd.x) > 4
如果向前走的速度大于4则触发.

(关于参数具体含义请查看"cns变量设置")
