FVar (math)

此触发器使用强制变量号作为参数.
将返回玩家指定的浮点型变量值.

格式:
FVar(表达式)

参数:
表达式
需要计算参数号的表达式.
目前有效号是0-39.

返回值类型:
浮点型

错误条件:
如果表达式计算出bottom,
或者表达式计算出无效的变量索引则返回bottom.

示例:
trigger1 = FVar(5) = -1.23
如果变量索引为5的浮点数 值为-1.23则触发.

特列:
绝大部分情况下可用Fvar来代替Var

注意:
在设置变量时,变量不能以大写字母开头,例如:
[State ]
type = VarSet
trigger1 = 1
FVar(0) = 0 ;这是错误的写法

[State ]
type = VarSet
trigger1 = 1
fvar(1) = 0 ;这是正确的写法
