Var (math)

此触发器使用强制变量号作为参数.
将返回玩家指定的整型变量值.

格式:
Var(表达式)

参数:
表达式
需要计算参数号的表达式.
目前有效号是0-59.

返回值类型:
整型

错误条件:
如果表达式计算出bottom,
或者表达式计算出无效的变量索引则返回bottom.

示例:
trigger1 = Var(0) = -34
如果索引为0的整型变量 值为-34则触发.

注意:
在设置变量时,变量不能以大写字母开头,例如:
[State ]
type = VarSet
trigger1 = 1
Var(0) = 0 ;这是错误的写法

[State ]
type = VarSet
trigger1 = 1
var(1) = 0 ;这是正确的写法
