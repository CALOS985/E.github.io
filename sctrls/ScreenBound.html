ScreenBound

指定玩家的运动是否被屏幕约束.
也决定着摄像机是否应该跟随玩家运动.
此控制器产生的结果只在1帧内有效.

必要参数:
无

可选参数:
value = bound_flag (布尔型)
如果bound_flag是0,玩家将被允许离开屏幕.
如果是1,则玩家将被约束在屏幕内.缺省默认是0.

movecamera = move_x_flag, move_y_flag (布尔型, 布尔型)
如果是1,指定摄像机应该跟随玩家分别在x和y方向平移.缺省默认都为0.

示例:
无

实例:
[State 0, ScreenBound]
type = ScreenBound
trigger1 = 1
value = 1
movecamera = 0,0
不允许人物离开屏幕也不允许镜头随人物移动
有时候也可以搭配width使用
