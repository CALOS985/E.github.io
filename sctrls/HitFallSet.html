HitFallSet

当玩家被击中,设置玩家的下落变量.

必要参数:
无

可选参数:
value = fallset_flag (整型)
如果fallset_flag是-1,则此控制器不会改变玩家是否将要下落.
如果fallset_flag是0,意味着玩家将不进入下落状态,1表示将进入下落状态.默认值为-1.

xvel = x_velocity (浮点型)
见下方.

yvel = y_velocity (浮点型)
若指定,分别设置玩家的fall.xvel和fall.yvel参数.
详见HitDef对这些参数的描述.

示例:
无

特例:
可作为特殊效果使用,比如对方空中被冰冻着地后再被非fall攻击不会被击飞.
[State 0, HitFallSet]
type = HitFallSet
trigger1 = time = 0
value = 0
