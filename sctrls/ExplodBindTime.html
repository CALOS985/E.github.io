ExplodBindTime

改变玩家的explod位置绑定时间.

必要参数:
无

可选参数:
ID = id_no (整型)
仅当explod拥有的ID号等于id_no时才能影响他们的位置绑定.
设置ID为-1来影响所有的explod.
默认值为-1.

time = binding_time (整型)
指定explod应该被约束到绑定点的时间帧数.(explod被创建时定义)
默认为1帧.time为-1时explod将被永久绑定或者直到另外的控制器改变绑定时间.

替代语法:
value = binding_time
也许可以用来代替time = binding_time.
